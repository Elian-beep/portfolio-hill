## Rodando localmente

Clone o projeto

```bash
  git clone [link_do_clone_portfolio-hill]
```

Entre no diretório do projeto

```bash
  cd portfolio-hill
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run dev
```
